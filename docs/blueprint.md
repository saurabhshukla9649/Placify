# **App Name**: Placify

## Core Features:

- Resume Data Extraction: Allow users to upload their resume (PDF/DOC) and automatically extract key information such as skills, projects, education, and experience for analysis.
- Placement Probability Score: Generate a dynamic 'Placement Probability Score' (0–100%) by analyzing extracted resume data, comparing against skills demand, and considering user-provided CGPA and certifications, using an AI tool.
- Resume Analysis & Suggestions: Analyze uploaded resumes for ATS compatibility, generate an ATS score, highlight missing keywords and formatting issues, and suggest improvements with practical examples using an AI tool.
- Communication Analyzer: Enable microphone access to record user speech, then analyze fluency, confidence, and clarity. Provide a communication score and actionable feedback, powered by an AI tool.
- Skill Gap Identifier: Compare user skills extracted from resumes or profiles against requirements for specified job roles (e.g., Data Analyst, Java Developer), identify missing skills, and recommend tailored learning paths with an AI tool.
- Interactive Dashboard: A personalized dashboard to visually display the Placement Probability Score, Resume score, Communication score, and Skill Gap Analysis results using modern charts, progress bars, and cards.

## Style Guidelines:

- The color scheme is predominantly light, chosen to evoke a sense of clarity, trust, and aspiration suitable for an educational and career-focused platform. The primary accent color is a clear, professional blue (#269CDA), symbolizing intelligence and reliability. The background color is a very subtle blue-gray (#F0F3F5), maintaining a clean and approachable canvas. A vibrant accent color, a rich purple-blue (#6A52E0), is used sparingly for highlights and calls to action, creating visual distinction and energy.
- For a modern and tech-inspired aesthetic, 'Space Grotesk' (sans-serif) will be used for headlines, conveying a sophisticated and analytical feel. 'Inter' (sans-serif) will serve as the primary body font, ensuring excellent readability and neutrality across all content.
- Clean and minimalistic line icons, or subtly filled icons, will be used. They should visually represent concepts of data analysis, progress, achievement, and communication, maintaining consistency with the app's modern and precise aesthetic.
- The layout will emphasize clarity and scannability, utilizing a card-based system within a flexible grid structure for displaying various scores and analyses. Content will be well-segmented to prevent cognitive overload, with multi-column arrangements for larger screens and streamlined single-column views for mobile devices to ensure full responsiveness.
- Subtle and purposeful animations will enhance user experience, including smooth transitions for data updates (e.g., score changes), elegant hover effects for interactive elements, and progressive filling animations for progress bars and charts, adding a dynamic and polished feel.
